Listado de artículos para la universidad

Imprescindibles

  1. Ordenador portátil: Es imprescindible para poder realizar trabajos y tareas en cualquier lugar.
  2. Papel: Para poder tomar apuntes o hacer ejercicios rápidos
  3. Bolígrafos y lápices: Para poder escribir en la libreta.
  4. Mochila: Para poder llevar todos los materiales a la universidad.
  5. Teléfono celular: Para poder hacer cualquier actividad en línea.

No imprescindibles

  • Calculadora: Para poder realizar cálculos matemáticos.
  • Libreta: Para poder tomar apuntes en clase.
  • Audífonos: Para poder ver vídeos educativos en la universidad.
  • Mouse: Para complementar el uso del computador.


Cualidades que see necesitan para ser un estudiante universitario exitoso

Responsabilidad:
Es importante ser responsable con las tareas y trabajos que se deben realizar.
Organización:
Es importante tener una buena organización para poder cumplir con todas las tareas y trabajos.
Disciplina:
Es importante tener disciplina para poder cumplir con los horarios y tareas establecidas.
Constancia:
Es importante ser constante en el estudio y en el trabajo para poder alcanzar los objetivos.
