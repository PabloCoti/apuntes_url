# Endomarketing

Tomando como base el marketing, pero hacia adentro de la empresa, par los trabjadores, proveedores y accionistas.

Dentro del endomarketing el enfoque no es vender el producto al trabajador que lo produce si no que se le vende como objetivo la empresa


| Marketing general   | Marketing interno         |
| ------------------- | ------------------------- |
| Cliente externo     | Trabajador                |
| Producto o servicio | Empresa                   |
| Técnica de ventas   | Comunicación interna      |
| Fuerza de ventas    | Equipo directivo          |
| Objetivo            | Incrementar la motivación |
